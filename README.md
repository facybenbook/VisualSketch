## 分支说明
- master 主分支/版本分支
- develop 最近开发分支，定期发布版本。
- package PackageManager包分支
- xxxx 开发中的功能分支，每个功能单独维护，完成后合并进入develop。

## 目录说明
- VisualSketch unity工程目录
- Image 展示图文件夹
- Doc 说明文档

## Git LFS
库中所有图片资源，prefab资源（虽然prefab可以使用文本格式，但是我们根本不会手动合并它），scene，等出文本以外的一切资源都使用LFS保存，各成员注意设置自己的git。
